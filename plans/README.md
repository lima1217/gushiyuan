## Findings considered and rejected

- Missing `/l/[streamId]` route: intentional while lineage is empty; issue #29
  explicitly removed stream pages until source-verified data returns.
- Unvalidated poem slug before filesystem join: static production serves
  pre-exported routes and does not execute that read for unknown paths; local
  dev hardening is too low leverage for this plan set.
- Missing CSP/HSTS: no concrete browser sink or sensitive authenticated surface
  was found; generic header advice is not an evidence-backed finding here.
- Linear body-search scan at 702 poems: algorithmically linear but no measured
  user-visible regression; revisit only with corpus growth or profiling.
- Volume-ambiguous React author keys: the pattern could collide, but no current
  catalog collision with demonstrated behavior was found.
- Dead `PoemRow` export and repeated small catalog scans: real cleanup
  opportunities, but too little impact to justify executor plans now.
